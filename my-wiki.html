二次函数图像与性质

来源 12 份资料 · 相关词条 5 个 · 上次更新 2 小时前

1. 抛物线的开口方向

二次函数 y = ax² + bx + c (a ≠ 0) 的图像是一条抛物线。 当 a > 0 时，开口向上；当 a < 0 时，开口向下。 [导入课件.pptx P3]

2. 对称轴

抛物线的对称轴是一条平行于 y 轴的直线，方程为： x = -b / (2a) [配套习题汇编.pdf P12]

3. 顶点坐标

顶点是抛物线的最高点或最低点，坐标为： ( -b/(2a), (4ac-b²)/(4a) ) [教学设计·二次函数.docx P5]

4. 增减性

当 a > 0 时，在对称轴左侧 y 随 x 增大而减小，右侧 y 随 x 增大而增大。 a < 0 时相反。 [海淀区 2026 一模卷 T6]

5. 与坐标轴交点

令 x = 0 得 y 轴交点 (0, c)； 令 y = 0 解 ax² + bx + c = 0 得 x 轴交点（个数取决于判别式 Δ）。 [配套习题汇编.pdf P15]
